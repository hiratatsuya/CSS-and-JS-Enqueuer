== Description ==

CSS and JS Enqueuer plugin will enable to enqueue  external CSS and JS, and edit it.
You can customize the web site at only the dashboard.

== Installation ==

1. Upload the `css-js-enqueuer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Open 'Appearance' -> 'Enqueue CSS JSâ€™, and edit it.
